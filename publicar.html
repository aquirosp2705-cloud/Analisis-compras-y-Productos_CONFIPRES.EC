Qué sube el sistema

Cliente nuevo: escriba el nombre del repositorio (ej. analisis-clientex) y use el botón verde: el sistema CREA el repositorio, ACTIVA GitHub Pages y PUBLICA — todo automático, sin entrar a GitHub. Con un token maestro guardado una sola vez, cada cliente nuevo es un solo clic.

Con un solo clic se publican 3 páginas en el repositorio: index.html (portal con la marca CONFIPRES.EC), dashboard.html (el dashboard interactivo completo) y base_productos.html (la base de productos clasificados, consultable con filtros). El cliente entra desde cualquier dispositivo a https://USUARIO.github.io/REPOSITORIO/. El token queda guardado en este equipo, así las siguientes publicaciones son de un solo clic.
